What is Metaleaf

Metaleafとは

MetaLeafは、バーチャル空間で学習者同士が自由に集まり、
「共に歩む」ためのオンラインコミュニティです。

初心者から上級者まで誰でも歓迎！

同じような境遇の人との繋がりが持ててモチベーションUP
時間管理や効率的な学習方法など一緒に頑張る仲間の生の声を聞けます。

GUIDE

Metaleafでできること

  • 始めたばかりで使い方がわからない...

    名前の前に😢マークをつけて初心者専用部屋に行ってみよう！

    「初心者専用部屋」に行くと、同じ境遇の仲間もできるし、優しい先輩方が話しかけてくれるよ！

    基本的な操作方法はこちら

  • 本業で疲れ切って寝てしまい、毎日作業する習慣がつかない...

    もくもくブース、もくもく部屋を使おう！

    毎日ログインする習慣から

    ここでは一人で集中して作業ができます。周りに仲間がいるから頑張ろうと思えるよ！

  • 一人で勉強していると、すぐにスマホ見ちゃう...

    ポモドーロ・タイムで集中モード！

    25分間の集中タイムに参加

    静かな空気に乗って、自分も自然と集中！

  • メタリーフ参加したばかりで馴染めない。
    質問したいけど話しかけられないな...

    「初心者交流会」を活用しよう！

    初心者交流会

    ( 毎週木曜日22:00〜 )

    講師の先生にコードのことや困っていることをきけます！

  • 技術的なことでつまずいて、質問したいな…

    レベル別質問 部屋を活用しよう！

    レベル別質問部屋

    ( 毎週月曜22:00〜 )
    レベル別で講師の先生に気軽に質問できます。

  • 案件獲得 部屋を活用しよう！

    案件獲得 部屋を活用しよう！

    案件獲得部屋

    ( 毎週月曜22:00〜 )
    専門の講師に気軽に質問できます。 他の人の話も聞くことで、いろいろな情報が得られます！

MetaLeafでは、学習者や講師たちと交流を深めるためのイベントやスキルアップを図れるイベントが多数開催されています

  • 01

    オンライン飲み会

    普段話す機会がない人とも交流ができ、モチベーションUP！

    月に１回程度、定期的に開催されています。

  • 02

    特別セミナー

    現役エンジニアやスクール講師が登壇するセミナーを開催

    コーディングのポイントや案件獲得に向けたマインドセット、 AI活用など、様々なテーマを取り上げて行われています。

  • 03

    ハッカソン

    チームで限られた期間内に１つの成果物を協力して作成

    複数人でデザインやコードを共有する方法を学び、他の人のコードを読む練習、進捗管理を経験することができます。
